# Contributing to Banglish-to-Bangla

Thank you for your interest in improving the Banglish-to-Bangla converter! We welcome contributions from the community.

## How to Contribute

1. **Reporting Bugs**: Use the GitHub Issue tracker to report bugs. Please include steps to reproduce the issue and the Banglish word that failed.
2. **Feature Requests**: Open an issue to suggest new features or improvements (e.g., support for specific dialects or complex conjuncts).
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature.
   - Ensure your code follows the existing style.
   - Submit a PR with a clear description of your changes.

## Development Setup

1. Install Python 3.12 or higher.
2. Ensure you have the `data/` folder with all required `.csv` and `.txt` files.
3. Run `python converter.py` to test your changes.
